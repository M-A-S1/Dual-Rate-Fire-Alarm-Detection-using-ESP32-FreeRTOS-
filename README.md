# Dual-Rate-Fire-Alarm-Detection-using-ESP32-FreeRTOS-
A real-time **fire alarm sound detection system** built on the **ESP32** using **FreeRTOS**, **I2S audio capture**, and **dual-window FFT analysis**.   The system detects the characteristic fire-alarm frequency (~3100 Hz) with both **fast response** and **high frequency accuracy**, and publishes results via a **WiFi web dashboard**.
